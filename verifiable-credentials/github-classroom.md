# Github Classroom

In this article you will learn how to use GitHub classroom to complete coding assignments.

## Create a Github Account

- Sign-up for an [Account](https://github.com/) on Github
- Login to your GitHub account

## Accepting an Assignment

- You will be provided with a link to accept and assignment on Github classroom
- Open the link in a web browser
- Click on the button to 'Accept this assignment'
- Refresh the page and click on the link to view your assignment repository

## Clone the Assignment Repository in Visual Studio

- Open Visual Studio on your desktop PC
- In the 'Get started' page, click the 'Clone a repository' selection
- In the GitHub assignment repository, click on the 'Code' button and copy the 'Clone' section copy the 'HTTPS' link
- Copy the link into the Visual Studio 'Repository location' text box 
- Create a local path to save your cloned repository and select it in the 'Path' text box
- Click the 'Clone' button when you are done

## Open the Cloned Repository in Visual Studio

- In 'Solution Explorer', click on the '.sln' file
- You can now start completing your assignment in the solution

## Completing the Assignment in Visual Studio

- In the Visual Studio solution, add your code to complete the assignment
- In the 'Git Changes' tab and enter a comment for your commit, then click the 'Commit All' link
- Use the 'circle' con to sync the commit with your remote GitHub assignment repository
- Verify, your commit is saved on GitHub by opening the solution folder in GitHub online assignment repository and checking your file changes
- If you make additional changes to your code, you can make multiple commits

## Submit your Complete Assignment

- When you have complete your assignment, open the assignment repository on GitHub online
- In the GitHub assignment repository, click on the 'Pull requests' link
- Open the 'Feedback' pull request
- In the 'Comments' section, type 'Assignment Completed' and click the 'Comment' button
- Your will receive feedback on your assignment
- You can carry out a discussion with assessor in the Pull Request comments section

## Seeking Clarification from your Assessor

- You can use the 'Feedback' Pull Request comments section to seek clarification from your assessor on the assignment and its instructions
